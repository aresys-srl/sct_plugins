# Plugin

This python package is a plugin designed for **Sar Calibration Toolbox (SCT)** to allow quality processing on EOS-04 L1
products, both L1-A and L1-B.

==Plugin name:== ``sct-eos04-reader``

The main purpose of this package is to provide an SCT input protocol compliant object properly wrapping and composing
the EOS-04 product info allowing a seamless processing of this product format for every analysis integrated in the tool.

Supported EOS-04 acquisition modes:

- **SCANSAR**
