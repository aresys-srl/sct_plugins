# SCT Plugin: NovaSAR1 format reader

Sar Calibration Toolbox (SCT) plugin for reading NovaSAR1 SAR L1-A (SLC) and L1-B (GRD) products.

Supported acquisition modes:

- **Stripmap**
- **Scansar**

This package and its dependencies can be installed using ``pip`` tool:

```bash
    pip install sct_novasar1_reader
```
