# SCT Plugin: ICEYE format reader

Sar Calibration Toolbox (SCT) plugin for reading ICEYE SAR L1-A (SLC) and L1-B (GRD) products.

Supported acquisition modes:

- **Stripmap**
- **Spotlight**
- **Scansar**

This package and its dependencies can be installed using ``pip`` tool:

```bash
    pip install sct_iceye_reader
```
