# SCT Plugin: Saocom format reader

Sar Calibration Toolbox (SCT) plugin for reading Saocom SAR L1-A (SLC) and L1-B (GRD) products.

Supported acquisition modes:

- **Stripmap**
- **Topsar**

This package and its dependencies can be installed using ``pip`` tool:

```bash
    pip install sct_saocom_reader
```
