# SCT Plugin: Aresys Products format reader

Sar Calibration Toolbox (SCT) plugin for reading Aresys SAR L1-A (SLC) and L1-B (GRD) internal products.

Supported acquisition modes:

- **Stripmap**
- **Spotlight**
- **Scansar**
- **Topsar**

This package and its dependencies can be installed using ``pip`` tool:

```bash
    pip install sct_aresys_reader
```
