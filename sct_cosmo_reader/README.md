# SCT Plugin: COSMO-SkyMed format reader

Sar Calibration Toolbox (SCT) plugin for reading COSMO-SkyMed SAR L1-A (SLC) and L1-B (GRD) products.

Supported acquisition modes:

- **Stripmap**
- **Spotlight**
- **Scansar**

This package and its dependencies can be installed using ``pip`` tool:

```bash
    pip install sct_cosmo_reader
```
