# SCT Plugin: ENVISAT/ERS ASAR format reader

Sar Calibration Toolbox (SCT) plugin for reading ENVISAT/ERS SAR L1-A (SLC) and L1-B (GRD) ASAR products.

Supported acquisition modes:

- **Stripmap**
- **Scansar**

This package and its dependencies can be installed using ``pip`` tool:

```bash
    pip install sct_asar_reader
```
