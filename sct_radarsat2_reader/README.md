# SCT Plugin: RADARSAT-2 format reader

Sar Calibration Toolbox (SCT) plugin for reading RADARSAT-2 SAR L1-A (SLC) and L1-B (GRD) products.

Supported acquisition modes:

- **Stripmap**
- **Spotlight**
- **Scansar**

This package and its dependencies can be installed using ``pip`` tool:

```bash
    pip install sct_radarsat2_reader
```
