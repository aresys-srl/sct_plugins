# SCT Plugin: Sentinel-1 format reader

Sar Calibration Toolbox (SCT) plugin for reading Sentinel-1 A/C/D SAR L1-A (SLC) and L1-B (GRD) SAFE products.

Supported acquisition modes:

- **Stripmap**
- **Topsar**

This package and its dependencies can be installed using ``pip`` tool:

```bash
    pip install sct_sentinel1_reader
```
