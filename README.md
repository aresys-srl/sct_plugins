# SCT Product Formats Plugins

SAR Product Formats plugins collection for SCT.

Supported products:

- **Sentinel-1 A/C**
- **NovaSAR1**
- **Saocom**
- **ICEYE**
- **RADARSAT-2**
- **EOS-04**
- **COSMO-SkyMed**
- **ENVISAT/ERS ASAR Format**
- **Aresys Product Format**

## Installation

Each plugin is standalone and can be installed using ``pip``. When installed, SCT can automatically discover and use it
for analyses.

### Sentinel-1

```bash
    pip install sct_sentinel1_reader
```

### NovaSAR1

```bash
    pip install sct_novasar1_reader
```

### Saocom

```bash
    pip install sct_saocom_reader
```

### ICEYE

```bash
    pip install sct_iceye_reader
```

### RADARSAT-2

```bash
    pip install sct_radarsat2_reader
```

### EOS-04

```bash
    pip install sct_eos4_reader
```

### COSMO-SkyMed

```bash
    pip install sct_cosmo_reader
```
