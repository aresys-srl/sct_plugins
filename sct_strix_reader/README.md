# SCT Plugin: Synspective StriX format reader

Sar Calibration Toolbox (SCT) plugin for reading Synspective StriX SAR L1-A (SLC) and L1-B (GRD) products.

Supported acquisition modes:

- **Stripmap**
- **Spotlight**

This package and its dependencies can be installed using ``pip`` tool:

```bash
    pip install sct_strix_reader
```
